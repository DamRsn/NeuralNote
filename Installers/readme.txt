You're about to install NeuralNote on your computer.

NeuralNote is an audio to MIDI plugin using Spotify's BasicPitch neural network for transcription.

Everything in NeuralNote is run locally on your computer. No audio data is sent to any server. In general, no data leaves your computer.

To get more information about NeuralNote, report issues, request features, contribute, or download the source code, see:
- https://github.com/DamRsn/NeuralNote

To get more information about BasicPitch, see:
- https://basicpitch.spotify.com/
- https://github.com/spotify/basic-pitch
